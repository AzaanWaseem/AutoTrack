# Frontend Requirements

## Node.js Requirements
- Node.js >= 18.0.0
- npm >= 9.0.0

## Installation
1. Install dependencies:
```bash
npm install
```

2. Create `.env` file with:
```
VITE_GOOGLE_CLIENT_ID=your_google_client_id
```

## Development
```bash
npm run dev
```

## Build
```bash
npm run build
```

## Core Dependencies
- React 18
- TypeScript
- Vite
- Tailwind CSS
- Axios
- React Router DOM
- Google OAuth
- Lucide React (icons)
